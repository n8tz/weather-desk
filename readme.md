<h1 align="center">widget-desk</h1>
<p align="center">Simple widget desk with weather </p>

## How to start

### using docker-compose

```
docker-compose up
```

& start localhost:8080

### using local setup

#### setup
```
npm i
```

#### dev
```
npm run start-dev
npm run runBrowser
```

#### prod
```
npm run build
npm run start
```
